# Personal CV Website

A professional, responsive personal CV website built with React and Vite. This site is designed to showcase my professional experience, skills, and projects to recruiters.

## Tech Stack
- **Frontend Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Planned)
- **Icons:** Lucide React (Planned)
- **Hosting:** GitHub Pages

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/koxiperu/koxiperu.github.io.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server:
```bash
npm run dev
```

### Deployment
The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
